ChemAxon licenses required for ACE

ACE needs two different licenses to use ChemAxon products: one for JChem Base, and one for MarvinJS.

  • Place the JChem license file in /home/tomcat9/.chemaxon/, and point to it a symlink license.cxl in the same directory.

  • Place the MarvinJS license code in $WEB/WEB-INF/epoch.properties.
