Public access ACE pages

  • General instructions
    • How to author questions in ACE.
    • Authoring and using textbooks in ACE.
    • Translating ACE into another language.
    • Using ACE for exams.
  • Authoring tools
    • ACE toDisplay() tester. See how ACE will display an expression that an author types into a question statement or feedback.
    • ACE structure comparator. See how ACE compares two structures.
    • JChem® structure comparator. See how JChem® (as opposed to ACE) compares two structures.
    • ACE substructure and skeleton comparator. See whether ACE finds that a structure contains a particular substructure or skeleton.
    • ACE conformer comparator. Compare two conformers with ACE.
    • ACE Standardizer. See how ACE standardizes structures that have multiple correct representations (such as ylides).
    • Lewis JS.
    • Get the data URL for the image of a Marvin structure.
  • Calculations
    • ACE Mechanism calculator. Calculate the products of a mechanistic step involving one or more compounds and electron-flow arrows.
    • ACE Pericyclic mechanism calculator. Determine whether any of the steps of a mechanism with electron-flow arrows are pericyclic.
    • ACE Parity calculator. Find the parity of each atom in a compound.
    • ACE Bond stereochemistry flags calculator. Find the stereochemistry flags of each bond in a compound.
    • ACE Functional-group finder. Find which functional groups a compound contains.
    • ACE Impossible synthetic starting material identifier. See whether ACE recognizes a compound as being an impossible compound to use as a starting material in a synthesis.
    • ACE contiguous C atom calculator. Find the largest set of contiguous C atoms in a compound.
    • ACE periodic table tester. Get various properties of an element.
    • ACE formula and unit canonicalization. See how ACE will convert units in a formula to their fundamental SI equivalents, and how it converts implicit to explicit multiplication.
    • ACE significant figure counter. Counts the significant figures in a number.
    • ACE mathematical expression calculator. See how ACE uses Maxima to reduce a mathematical expression to its simplest form.
  • Structure manipulation
    • ACE Rotation about bonds. Designate a particular dihedral angle for a bond in a 3D structure.
    • ACE Chair flip. Flip to the other chair conformer of a substituted saturated six-membered ring.
    • ACE Chair or half-chair flip. Flip to the other chair or half-chair conformer of a substituted saturated or monounsaturated six-membered ring. (For saturated compounds, we recommend the ACE Chair flip page instead of this one.)
    • ACE Stereocenter configuration inversion. Invert the configurations of selected sp3 stereocenters in a 2D or 3D structure, or get the structure's mirror image.
    • ACE Stereoisomer generation. Convert a compound with unspecified configurations at its stereocenters into all stereoisomers.
  • Private pages for developers.

A Web-based homework program that uses many of these tools can be found here.

Also available: a modified version of Walter Zorn's Javascript Graphics library. Walter Zorn passed away in 2009, and his scripts, though still available, are no longer being maintained or improved. We have made some additions to the script:

  • function drawArrow() allows one to draw arrows
  • functions setOnclick() and setOnmousemove() make the graphics elements "transparent" to mouse clicks and movements.
