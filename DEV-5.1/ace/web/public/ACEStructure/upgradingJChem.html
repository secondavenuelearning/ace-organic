Upgrading JChem

KeyMeaning
$BASE /home/aceorg/aceorg
$VERS the current version of ACE, e.g., DEV-3.2.
$WEB $BASE/$VERS/ace/web
  1. Go to the JChem download site, and download the platform-independent version of JChem (zip file).
  2. Unpack the JChem zip file into $BASE, and change the name of the parent folder jchem to include the version number.
  3. Unzip the file api-docs.zip in the newly downloaded JChem folder.
  4. Implement the new JChem license, if necessary.
  5. Add the necessary symlinks by running the following Perl programs found in $BASE/scripts:
    configureJChem.pl DEV-3.1 jchem5.4.1
    setIdentity.pl DEV-3.1
  6. If necessary, regenerate the JChem database tables.
  7. Recompile the source code.
  8. Use the admin tool to clear the Reactor cache (i.e., delete all rows in the JChem database table reactor_results_v4).
